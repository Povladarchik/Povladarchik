## Projects

**MathLLM** — *Automated Mathematical Solution Verification*
System that uses LLMs to generate and execute Python verification code for step-by-step mathematical solutions.
[GitHub](https://github.com/Povladarchik/MathLLM)

**Emotional LLM & Game Theory** — *Research*
Study of how emotional prompting affects LLM decision-making in the Centipede Game, benchmarked against human behavior and optimality criteria.
[GitHub](https://github.com/iLoveRamona/emotional_LLM)

**Price Alchemists** — *Avito Analyst Academy, Final Project (Data Science track)*
Computer vision application for object price estimation from photos. Combines image processing with regression modeling.
[GitHub](https://github.com/vainmoon/PriceAlchemists) · [Presentation](https://docs.google.com/presentation/d/1sfZfzeYIO2v0_R6OlkJXitiW7uSADPvzeDZHifmunX4/edit?usp=sharing)

ML pipeline for classifying hydrodynamic flow regimes (radial flow, bilinear flow, wellbore effects) in oil wells from pressure time series.
[GitHub](https://github.com/Povladarchik/siam-ml-hack)

**DrinkGenius Bot** — *Telegram Bot*
Cocktail recommendation system. Suggests drinks based on user preferences, displays recipes, and finds similar cocktails via ingredient matching.
[GitHub](https://github.com/Povladarchik/DrinkGenius-Bot)
